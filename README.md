# Building the code

This is a standard maven project, checkout the code and run:

```bash

mvn clean install

```

# Running the code

This project packages the application as an uberjar. To run it from the command line:

```bash

java -jar target/fizzbuzz-0.0.1-SNAPSHOT.jar

```
